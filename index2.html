Fonts to PNG - FA2PNG Alternative

What is F2PNG

F2PNG is a online tool to convert Font Awesome and Ion Icons into PNG images. It's an alternative to FA2PNG.io and other Font Icons to PNG services.

How does it work?

It's a very simple tool.
1. Select an icon, font size and color of your choice and press the "Generate Button". A preview will be displayed on the right.

2. Click the Download PNG button to save the image to your device.

What icon fonts are supported?

Currently F2PNG supports Font Awesome and Ion Icons. We will try to add more open source fonts in near future.

Why did you make it when FA2PNG and other services exist already?

I love those sites but they were built years ago, and are not maintained. I myself got frustated when they didn't work when I wanted to convert some icons to png and made this tool. It does not need a server to run and is hosted on Github pages so it won't go offline that often.
